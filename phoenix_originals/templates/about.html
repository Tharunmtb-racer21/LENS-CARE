LENS CARE

Interactive Optical Lens Advisor

🎯 What is Lens Care?

Lens Care is a comprehensive web-based optical prescription analysis system with user authentication, personal dashboards, and prescription tracking over years. Built with Flask, Excel storage, and modern web technologies.

✨ Features

User Management

  • Secure registration and login
  • Password hashing for security
  • 30-day persistent sessions
  • Doctor and patient roles

Eye Tests

  • Complete 10-question Color Blindness Test
  • 8-level Visual Acuity Test (20/200 to 20/15)
  • Color blindness type identification
  • Results automatically saved

Dashboard Analytics

  • 4 different Chart.js visualizations
  • Spherical Equivalent trend tracking
  • Sphere and Cylinder progression charts
  • Combined power visualization
  • Automatic trend analysis

Prescription Management

  • Add unlimited prescriptions
  • Track changes over years
  • Beautiful timeline view
  • PDF report generation
  • Excel data storage (persists after closing)

🔬 Physics Calculations

Spherical Equivalent

SE = Sphere + (Cylinder/2)

Combined refractive error for lens selection

Lens Power

Power in diopters where f is focal length in cm

🛠️ Technology Stack

Backend

  • Flask - Python web framework
  • Excel (openpyxl) - Data storage
  • Werkzeug - Security & hashing
  • ReportLab - PDF generation

Frontend

  • HTML5 & CSS3
  • JavaScript ES6
  • Chart.js - Data visualization
  • Sunset Gradient theme
